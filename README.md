# jQuery easing parabola
This script adds new easing to jQuery library.
## Usage
Here is an example of usage
```javascript
  $('.sun').animate({
        top: '45px'
    }, {
        'duration': 6000, //6 seconds
        'easing': 'parabola' //easing
    });
```
